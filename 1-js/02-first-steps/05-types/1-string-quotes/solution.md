
Зворотні лапки вбудовують вираз, що знаходиться у `${...}`, у рядок.

```js run
let name = "Ілля";

// вираз — число 1
alert( `привіт ${1}` ); // привіт 1

// вираз є рядком "ім'я"
alert( `привіт ${"ім'я"}` ); // привіт ім'я

// вираз є змінною, вбудовуйте її
alert( `привіт ${name}` ); // привіт Ілля
```
