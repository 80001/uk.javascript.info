# Діти у DOM

Подивіться на цю сторінку:

```html
<html>
<body>
  <div>Користувачи:</div>
  <ul>
    <li>Іван</li>
    <li>Петро</li>
  </ul>
</body>
</html>
```

Вкажіть принаймні один спосіб доступу до кожного з перелічених нижче DOM вузлів:
- До DOM вузла `<div>`?
- До DOM вузла `<ul>`?
- До другого `<li>` (Петро)?
